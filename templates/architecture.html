Key Features

üîê User Authentication

Secure login/registration with password hashing using Werkzeug. Session management with Flask-Login. User-specific data isolation.

üîç AI-Powered Search

Perplexica integration with multiple focus modes (web, academic, YouTube, Reddit). Falls back to SerpAPI for Google Search when needed.

üëÅÔ∏è Vision AI

GPT-4 Vision for intelligent image analysis. Describes content, extracts text, identifies objects. OCR fallback with Tesseract.

üìö Document RAG

Upload and analyze documents (PDF, Word, Excel, PowerPoint, images). Content extraction and indexing for retrieval-augmented generation.

üí¨ Multi-Mode Chat

Four response modes: Summarize (concise), Balanced (default), Verbose (detailed), Deep Thinking (analytical).

üì± Responsive Design

Mobile-first design with hamburger menu. Optimized for all screen sizes from phones to desktops.
